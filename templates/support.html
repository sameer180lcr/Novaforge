Getting started

  1. Create an account and choose a model.
  2. Get your API key from Dashboard → Keys.
  3. Use the quick-start templates below.

LLM Quick start (Python)

import requests
API_KEY = 'YOUR_KEY'
resp = requests.post('https://api.novaforge.ai/v1/chat', json={
  'model': 'llm-pro-70b',
  'messages': [{'role': 'user', 'content': 'Hello!'}]
}, headers={'Authorization': f'Bearer {API_KEY}'}).json()
print(resp['choices'][0]['message']['content'])

Image Generation (cURL)

curl -X POST https://api.novaforge.ai/v1/images \
 -H "Authorization: Bearer YOUR_KEY" \
 -H "Content-Type: application/json" \
 -d '{
   "model": "img-gen-studio",
   "prompt": "a cinematic portrait of a snow leopard, 85mm, shallow DOF"
 }'
